My project is Student Record Management System, which is used to maintain student records in a digital and efficient way. It helps reduce manual work and makes record management easier and error-free.The program is user-friendly and provides a menu-based interface, so users can easily choose the required option.For adding student records, I used a Singly Linked List (add_end) concept. Each record contains the student's roll number, name, and percentage.The project also allows users to delete and modify records using the roll number, name, or percentage, making it flexible and convenient.I have implemented a Save function to store all the updated records permanently and a Display function to view the saved records.Finally, when the user exits the program, it asks whether they want to save the changes before exiting or exit without saving, giving the user full control over their data.
